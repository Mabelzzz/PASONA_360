# How to Run the Project

## Install Dependencies
```bash
npm i
```

## For Backend
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm i
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## For Frontend
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm i
   ```
3. Start the frontend server:
   ```bash
   npm start
   ```
